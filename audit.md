# Technical Audit Report: Aegis Business Growth OS

Aegis Business Growth Operating System is an enterprise-grade AI Command Center and multi-agent boardroom simulation designed to analyze business financials, marketing, customer success, and sales funnels to formulate 12-month blueprints.

## 2. Technology Stack & Packages Used

### Frontend (Client-side Next.js)
*   **Framework**: Next.js 14 (App Router) — Server and client component optimization.
*   **Styling & UI**: TailwindCSS — Modern professional custom white and orange theme layouts.
*   **Icons**: `lucide-react` — Slick vector icon representations.
*   **Animations**: `framer-motion` — Smooth step-by-step card animations.
*   **Data Visualizations**: `recharts` — Responsive interactive area, bar, and conversion funnel charts.

### Backend (Server-side API)
*   **Runtime**: Node.js with TypeScript — Enforces type-safe model validations.
*   **HTTP Server**: Express.js — Centralized API route routers.
*   **Database ORM**: Prisma ORM — Schema mapping and type-safe database queries.
*   **Database Engine**: SQLite — Local file database connector (`dev.db`) enabling zero-configuration workspace loading.
*   **AI Integration**: Official `@google/generative-ai` SDK — Interface calls directly to Google Gemini 1.5 Flash.
*   **Input Validations**: Zod — Middleware request body schema validators.
*   **Authentication**: JSON Web Token (`jsonwebtoken`) & `bcryptjs` — User encryption and JWT-secured organization access.

## 3. Core Features & Capabilities

1.  **Zero-Config SQLite Database**: The system operates locally out-of-the-box. Pushing schemas or migrations compiles instantly.
2.  **6-Step Discovery Onboarding Wizard**: Gathers essential startup indicators (HQ, Category, monthly financials, product details, target audience, and bottlenecks).
3.  **PDF/CSV Auto-Fill Engine**: Users can drop PDF pitch decks or CSV spreadsheets. The frontend dynamically parses row logs, displaying a spreadsheet grid preview, and auto-populates onboarding variables.
4.  **Generative AI Boardroom Specializations**: Generates analysis loops across Finance, Marketing, Sales, Customer Success, Strategy, and CEO units.
5.  **Robust Dual-Mode Fallback**: Next.js is configured to route calls to the Express API. If the API is offline or database configuration errors out, it falls back to the client-side sandbox simulation.
6.  **Concurrently Orchestration Launcher**: Run `npm run dev` in the root folder to spin up Next.js on port `3000` and Express on port `5000` concurrently in a single console terminal.
